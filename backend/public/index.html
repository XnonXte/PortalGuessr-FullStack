List of available endpoints

/chambers

  1. GET /chambers - get all chambers.
  2. GET /chamber/<id> - get a specific chamber. The id being the MongoDB collection ID.
  3. GET /chambers/random/<amount> - get an amount number of chambers (random difficulty).
  4. GET /chambers/random/<amount>/<difficulty> - get an amount number of chambers with difficulty difficulty.
  5. POST /chambers/new - create a new chamber collection.
  6. PATCH /chambers/<id> - edit a chamber collection.
  7. DELETE /chambers/<id> - delete a chamber collection.
